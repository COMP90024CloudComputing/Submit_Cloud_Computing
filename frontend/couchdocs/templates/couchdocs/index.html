Technologies

In our system, many cluster and cloud computing technologies are utilized.

Nectar

Nectar ( The National eResearch Collaboration Tools and Resources project) is a Australian Research Cloud plantform.

Hadoop

The Apache™ Hadoop® project develops open-source software for reliable, scalable, distributed computing.

Spark

The Apache™ Spark provides programmers with an application programming interface centered on a data structure .

Couch DB

CouchDB is open source database that focuses on ease of having an architecture that "completely embraces the Web".

Nginx

NGINX accelerates content and application delivery, improves network security, facilitates availability and scalability for the busiest web sites on the Internet.

SU coreNLP

CoreNLP provides a set of natural language analysis tools. It can give the base forms of words, their parts of speech, whether they are names of companies.

AURIN

AURIN.Browse the growing list of all datasets accessible through the AURIN Portal. Over 1,200 datasets from over 40 sources covering a wide range of disciplines.

Machine Learning

Machine learning is a type of artificial intelligence (AI) that provides computers with the ability to learn without being explicitly programmed and obtain new information.
